# SmartLab-IoT-Project
 ⚡ SmartLab – IoT Energy Management System

📖 Overview
SmartLab is an IoT-based energy management system designed to optimize energy usage in computer laboratories. The system automatically controls lights, computers, and door access based on motion detection to reduce unnecessary energy consumption.

## 🚀 Features
- Motion sensors detect user presence in the lab
- Automatically turn lights and computers ON when the lab is occupied
- Turn OFF devices when the lab becomes empty
- Automatic door locking and unlocking
- Helps reduce energy waste and improve efficiency

 🛠 Technologies Used
-  ESP32 microcontroller
-  Motion sensors
-  Relay modules
-  IoT communication
-  Basic hardware integration

🎯 Project Goal
The main goal of this project is to demonstrate how IoT technologies can be used to create smart environments that save energy and improve resource management in educational institutions.
